Простое сообщение с интерфейсом

Создай интерфейс Messenger, который содержит один метод Message() string, возвращающий строку с сообщением. Создай структуру BasicMessenger без полей, реализующую интерфейс Messenger. Метод Message() должен возвращать фиксированную строку, например, "Я сообщение!".

В функции main:
Создай объект BasicMessenger.
Вызови метод Message() напрямую и выведи результат с помощью fmt.Println.

Пример вывода
Я сообщение!
